# EXCEL Quick Plotter（Excel 选区一键作图）

一个基于 **PyQt5 + Matplotlib** 的桌面悬浮工具：在 Excel 中框选数据后，点击按钮或按全局快捷键，即可把图表追加到“多图画板”中；如果未检测到 Excel 选区，会自动回退到**剪贴板表格解析**（适用于 Google Sheets / 网页表格 Ctrl+C）。

> 主要面向 Windows + Microsoft Excel（xlwings 通过 COM 读取活动 Excel 选区）。

## ✨ 主要功能

- **悬浮窗**：无边框、可拖拽、边缘磁吸；鼠标移出自动半透明；顶部区域双击可折叠/展开。
- **数据获取**：
  - 优先读取“活动 Excel 实例”的当前选区。
  - 读取失败时自动解析剪贴板（Tab 分隔表格为主，简单 CSV 为辅）。
- **5 种图表类型（下拉选择）**：
  - **Box Plot**：每列一组数据；散点 + 箱线 + 横置直方图 + 统计值；支持“离群点”高亮开关。
  - **Scatter (双组)**：至少 2 列 (X,Y)；若存在 4 列，会尝试作为两组 (X1,Y1) + (X2,Y2)。
  - **Scatter (多组)**：每两列为一组 (X,Y)，可扩展到任意组数。
  - **Heatmap**：把选区当作数值矩阵；非数值会被转换为缺失并以灰色遮罩。
  - **Line Plot (多行)**：每一行绘制一条折线；可自动识别第一行作为 X 轴标签；缺失值会形成断点并加强显示。
- **多图画板**：图表会按“向下堆叠”追加；支持单图删除、批量导出、清空画板、画板置顶。
- **复制高清原图**：每张图的工具栏包含“复制图片”；也可在图上双击快速复制高清 PNG 到剪贴板（适合粘贴到 PPT）。
- **悬浮提示**：多数图表支持鼠标悬浮显示数值与行号（依赖 `mplcursors`）。

## 🧩 环境与依赖

- Python：建议 **Python 3.8+**（3.7 理论可用，但不再推荐）。
- 系统：Windows（读取 Excel 活动选区依赖 COM）。
- Excel：建议使用 Microsoft Excel。
- 依赖安装：

```bash
pip install -r requirements.txt
```

> 提醒：如果 Excel 以“管理员权限”运行，而 Python 以普通权限运行（或反过来），COM 可能无法访问活动 Excel 实例，导致读取失败。此时可以让两者保持同一权限级别，或直接使用“剪贴板回退”方案。

## 🚀 运行方式

```bash
python main.py
```

运行后会出现悬浮窗 **“EXCEL快速分析”**。

## 🧭 使用说明（推荐流程）

1. 在 Excel 中框选数据区域。
2. 在悬浮窗右上角下拉选择图表类型（默认 Box Plot）。
3. 点击 **“✨ 提取并作图”**，或按 **全局快捷键：左 Alt + K**。
4. 图表会追加到“数据分析画板”窗口中：
   - 顶部按钮：**💾 批量导出 / 🗑️ 清空画板 / 📍 置顶**。
   - 每张图卡片：标题可编辑（导出文件名来自标题），右侧 **×** 可删除该图。
   - 图表工具栏：缩放/平移/保存 + **复制图片**。

## 📌 选区格式约定（非常重要）

- **Box Plot**：
  - 每一列视为一组；允许第一行是表头（程序会自动检测）。
  - 非数值会被忽略；“离群点”按 IQR(1.5×IQR) 规则高亮。
- **Scatter (双组)**：
  - 至少 2 列：第 1~2 列为 (X,Y)。
  - 若存在 4 列：第 3~4 列会尝试作为第二组 (X,Y)（若无有效数据则自动退化为单组）。
- **Scatter (多组)**：
  - 每两列一组：(1,2) 为 Group1，(3,4) 为 Group2 ...；最后落单的一列会被跳过。
- **Heatmap**：
  - 把选区当作矩阵；支持常见格式（如 `1,000`、`12%`），非数值会变为缺失。
  - 小矩阵会自动在格子中标注数值（默认阈值 400 个格）。
- **Line Plot (多行)**：
  - 每一行是一条曲线；每一列对应一个 X 轴分类。
  - 第一行若包含非数字内容，会被视为 X 轴标签行。

## ❓ 常见问题

- **提示“xlwings 未检测到活动 Excel 实例”**
  - 常见原因：Excel/脚本权限不一致、使用 WPS、或 Excel 不在同一用户会话。
  - 解决思路：让 Excel 与程序使用相同权限启动；或使用剪贴板回退（框选后 `Ctrl+C`）。

- **全局快捷键不可用**
  - 程序依赖 `keyboard` 或 `pynput` 注册全局快捷键；某些环境可能被安全策略拦截。
  - 解决思路：确认已安装依赖；不行就直接点击按钮触发。

- **批量导出文件名奇怪 / 导出失败**
  - 批量导出的文件名来自图卡片标题，并会过滤掉不适合作为文件名的字符；建议先改一个简短标题再导出。

## 📦 打包（PyInstaller）

- **目录分发（onedir）**：
  ```bash
  pyinstaller EXCEL-Quick-Plotter.spec
  ```

- **单文件（onefile）**：
  ```bash
  pyinstaller build_onefile/EXCEL-Quick-Plotter.spec
  ```

打包配置已包含 `icon.ico` 与 `style.qss`，以及 `keyboard/pynput/mplcursors` 等隐藏依赖。

## 📁 项目文件

- `main.py`：程序入口（悬浮窗、取数线程、全局快捷键、画板管理）。
- `box_plot.py` / `scatter_plot.py` / `scatter_plot_multi.py` / `heatmap_plot.py` / `line_plot.py`：各类图表渲染逻辑。
- `numeric_coercion.py`：数值清洗与强制转换工具。
- `style.qss`：Qt 样式表。
- `EXCEL-Quick-Plotter.spec`：PyInstaller onedir 打包配置。
